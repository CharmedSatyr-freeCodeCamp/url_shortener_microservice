{{#home}}

Directions

To save a new address, add the full URL you'd like shortened, e.g., https://example.com/ to the end of https://charmed-urls.herokuapp.com/new/.

A shortened URL that redirects to your original URL will appear on your screen.


For example, https://charmed-urls.herokuapp.com/new/https://example.com/

returns https://charmed-urls.herokuapp.com/0f115/,

which will redirect to https://example.com/.
